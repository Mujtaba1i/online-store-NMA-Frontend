# online-store-NMA-Frontend
NMAZONE is a full-stack e-commerce web application that supports both seller and customer. The platform allows sellers to manage product listings, while customers can browse products, add items to a cart, and place orders through a simple and intuitive interface.

The application includes secure authentication with separate dashboards for sellers and customers, ensuring role-based access and functionality. Sellers can add, edit, and delete products, while customers can view product details, manage their cart, and complete the checkout process.

## Features:

- Sign up for a seller and customer account.

- Sign in.

- Seller can add and delete products.

- Customer can view products.

- Cart functionality.

## User Stories:  

1. As a seller, I want to create a new account. 

2. As a seller, I want to log in to the website. 

3. As a seller, I want to log out from the website. 

4. As a seller, I want to add products to display on the website. 

5. As a seller, I want to edit added product.

6. As a seller, I want to delete added products.

7. As a customer, I want to create a new account. 

8. As a customer, I want to log in to the website. 

9. As a customer, I want to log out from the website. 

10. As a customer, I want to see added products with their descriptions.

11. As a customer, I want to add products to cart.

12. As a customer, I want to checkout from cart and place order.
